Learn JS and jQuery Terms

JavaScript (JS)

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

Arithmetic operators are used to perform arithmetic between variables and/or values. Examples: +, -, *, /, and %.

Variables

Variables are containers for storing data values.

Functions

A block of code designed to perform a particular task.

Methods

Actions that can be performed on objects. A method is a function that belongs to an object.

Arguments

The real values passed to the function. Not the same as parameter!

Parameters

Names listed in the function's definition.
